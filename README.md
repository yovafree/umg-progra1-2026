# Programación I - UMG 2026

Repositorio de ejemplos y ejercicios del curso de **Programación I** de la Universidad Mariano Gálvez de Guatemala.

## Tecnologías

- **Lenguaje:** C#
- **Framework:** .NET 9.0
- **IDE:** Visual Studio / VS Code

## Estructura del repositorio

```
umg-progra1-2026.sln          ← Solución principal
│
├── Semana 2/
│   ├── ejemplo01/             ← Variables, tipos de datos, entrada/salida por consola
│   └── ejemplo02/             ← Estructuras condicionales (if/else)
└── ...
```

Cada semana contiene proyectos de consola independientes que refuerzan los temas vistos en clase.

## Requisitos previos

1. [.NET 9.0 SDK](https://dotnet.microsoft.com/download/dotnet/9.0)
2. Un editor de código (Visual Studio 2022+ o VS Code con la extensión C# Dev Kit)

## Cómo ejecutar un ejemplo

```bash
# Desde la raíz del repositorio, ejecutar un proyecto específico:
dotnet run --project "Semana 2/ejemplo01"
```

O abrir `umg-progra1-2026.sln` en Visual Studio y seleccionar el proyecto de inicio deseado.
